Instructions

This part is about improving the character building in the game:

  • Adding more characteristics to each playable character
  • Adding more custom interaction between each character
  • Adding RPG-like level, class and race systems
Basic Characteristics
Requirements Difficulty Mod file
Base Game & Dynamic Characters Intermediate basic_characteristics.ts
  • Add an option “Character” in the combat options.
    • Displays a character file with its name, statistics, class and race (see classes.json and races.json).
  • If the character attacks a class or race it's strong against, it deals x2 damages (x4 if both the enemy's class and race are weak against the character's class and race) (see Damage Modifiers section).
  • If the character is attacked by a class or race it's weak against, it receives x2 damages (x4 if both the enemy's class and race are strong against the character's class and race) (see Damage Modifiers section).
  • If the character attacks a class or race it's weak against, it deals /2 damages (/4 if both the enemy's class and race are strong against the character's class and race) (see Damage Modifiers section).
  • If the character is attacked by a class or race it's strong against, it receives /2 damages (/4 if both the enemy's class and race are weak against the character's class and race) (see Damage Modifiers section).
  • A “strong” attack will be written as “Crushing hit”.
  • A “weak” attack will be written as “Glancing hit”.
Character Creation
Requirements Difficulty Mod file
Basic Characteristics Intermediate character_creation.ts
  • When starting the game, the player can create its own custom character.
  • The player can pick a class and race out of the different options in classes.json and races.json.
  • The statistics of the character can be attributed be the player. You get to decide how many maximum points the player can put in the statistics.
  • Once the character is completed, a quick summary is displayed and the player is asked for confirmation.
Basic Characteristics II
Requirements Difficulty Mod file
Basic Characteristics Intermediate basic_characteristics_2.ts
  • The character has a good or evil alignment based on its class (see classes.json).
  • The alignment is displayed in the character file of the “Character Option”.
  • The character unlocks the fighting option “Special Move”, and allows the character to use a special class skill once every fight. You get to choose these skills' effects.
Level and Experience
Requirements Difficulty Mod file
Basic Characteristics Easy level_and_experience.ts
  • The character randomly earns between 15 and 50 experience points after each victory.
  • The character gains a level each time it earns enough experience points. You decide of how much XP the character needs to level up.
  • Each time the character levels up, it randomly gains statistics. You decide what statistics it gains.
